# Metrics Analysis Agent

## Responsibility
Summarize campaign, ad group, targeting, and search-term performance from normalized monitoring snapshots.

## Metrics
Spend, sales, orders, impressions, clicks, CPC, CTR, CVR, ACOS, ROAS, and no-sales conditions.

## Outputs
Plain-language trends, data gaps, top spend waste, strong performers, and low-traffic opportunities.

## Boundary
The agent explains metric movement only. Recommendation types and statuses come from deterministic rules.
