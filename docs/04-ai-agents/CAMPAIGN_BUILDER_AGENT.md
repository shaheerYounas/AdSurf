# Campaign Builder Agent

## Responsibility
Explain generated campaign structure and help users review Hero, Exact, Phrase, Broad, budgets, bids, and groups.

## Inputs
Approved keywords, relevance scores, search volume, product defaults, campaign plan output.

## Prohibited
- Do not select Hero outside deterministic rule.
- Do not change group size outside 5 to 7 rule.
- Do not approve plan or export.

## Acceptance Criteria
Agent summary names the Hero keyword, group count, default budget/bid source, and negative keyword strategy.
