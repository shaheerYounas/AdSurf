# Monitoring Agent

## Responsibility
Summarize 14-day campaign performance and make metric trends understandable.

## Inputs
Monitoring snapshots with impressions, clicks, spend, sales, orders, CPC, CTR, CVR, and ACOS.

## Prohibited
- Do not create final optimization decisions without rule engine output.
- Do not change bids, budgets, or campaign states.

## Acceptance Criteria
Summaries identify metric direction, data gaps, and rule-triggered recommendation status.
