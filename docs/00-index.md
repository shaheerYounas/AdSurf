# Documentation Index

## Canonical Product Rule
AI agents do not directly execute dangerous changes. Rules calculate. AI explains. Humans approve. The system logs every decision.

## Documentation Maintenance
Future behavior-changing code changes must update the relevant docs in the same change. API, database, business rule, approval, AI, and workflow changes are not complete until documentation and tests are aligned.
