# Out Of Scope

## Not In MVP
| Item | Reason |
| --- | --- |
| Direct Amazon Ads API execution | Bulk sheets validate workflow before live automation risk. |
| Silent autonomous changes | Violates human approval principle. |
| Automatic budget increases | Budget changes require explicit approval. |
| Automatic campaign pauses | Pause recommendations require explicit approval. |
| Automatic negative keyword publishing | Negative recommendations require explicit approval. |
| Billing/subscriptions | Product workflow comes first. |
| Non-Amazon channels | Domain rules are Amazon Ads-specific. |
| Full BI warehouse | MVP stores required monitoring and audit data only. |

## Explicit Never-Automate Without Approval
- Live campaign creation.
- Bid changes.
- Budget changes.
- Campaign pauses.
- Negative keyword additions.
- Campaign locks.
- Bulk sheet export approval.
- Future Amazon Ads API execution.
