# Customer Language Guide

## Voice
Clear, calm, and specific. Customers should know what the system recommends, why it recommends it, and what happens only after approval.

## Preferred Terms
| Use | Avoid |
| --- | --- |
| Recommended | AI decided |
| Needs approval | Will automatically change |
| Rejected because relevance score is too low | Bad keyword |
| Bulk sheet export | Automated live launch |
| Campaign lock recommendation | Guaranteed winner |

## Approval Copy Requirements
Approval UI must state the object, impact, source rule, and whether the action is MVP export-only or future live execution.
